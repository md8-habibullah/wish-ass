# 🛡️ Wish Ass Client: Clinical Interface

The frontend interface for the **Wish Ass** medical procurement platform. Built with a focus on speed, security, and a premium "Zero Lag" user experience.

## 🚀 Tech Stack
- **Next.js 15+**: High-performance React framework.
- **Tailwind CSS & Framer Motion**: Advanced styling and micro-interactions.
- **Better Auth Client**: Secure session management.
- **TanStack Query**: Optimized data fetching and caching.

## 🛠️ Features
- **WACI AI**: Integrated clinical intelligence assistant.
- **Procurement Dashboard**: specialized views for Medical Staff and Pharmacists.
- **Order Tracking**: Real-time status updates for clinical requisitions.
- **Inventory Explorer**: Advanced search and categorization of medical supplies.

## 🏁 Getting Started
1. **Install Dependencies**:
   ```bash
   pnpm install
   ```
2. **Setup Environment**:
   Copy `.env.example` to `.env` and configure `NEXT_PUBLIC_API_URL`.
3. **Run Development**:
   ```bash
   pnpm run dev
   ```

Built for the **Wish Ass** network.
